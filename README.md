## Project Summary

Guess Game is an interactive browser game created with vanilla JavaScript, HTML5, and CSS3. Its presentation leans into a terminal-like layout, while the player enters guesses through an on-screen keypad and uses the feedback to work out the secret number.

The experience is built for short, fast rounds with a constrained number of attempts and a sharp neon visual style. Animated win and loss dialogs plus a rank system add a more arcade-like finish without making the rules difficult to learn.

## Script Features

- High-low hints after each guess
- Interactive keypad input for browser play
- Animated win and lose popups
- Rank system tied to performance
- Modern neon interface with terminal-style presentation
- Built with vanilla JavaScript, HTML5, and CSS3
